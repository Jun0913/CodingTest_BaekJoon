# 노란불 신호등

## 문제 핵심
여러 신호등이 반복되는 주기를 가질 때,  
모든 신호등이 **동시에 노란불이 되는 가장 빠른 시간**을 구하는 문제.

---

## 풀이 아이디어
각 신호등은 `[g, y, r]` 형태로 주어지고,  
한 주기 `cycle = g + y + r`를 기준으로 반복된다.

특정 시간 `t`에서 현재 위치는 다음과 같이 계산한다.

```java
int pos = (t - 1) % cycle + 1;
```
이때 g < pos && pos <= g + y이면 노란불이다.

시간을 1초부터 증가시키면서 모든 신호등이 노란불인지 확인하고,
처음으로 조건을 만족하는 시간을 반환했다.

## 어려웠던 부분
시간에 따라 상태가 반복된다는 점을 떠올리는 것
현재 시간을 주기 안의 위치로 변환하는 방법 (% 연산)

## 배운 점
반복되는 문제는 주기 + 나머지 연산으로 접근할 수 있다
복잡한 문제는 **단일 요소(신호등 1개)** 부터 생각하면 풀기 쉬워진다
